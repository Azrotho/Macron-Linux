# Branding de la Distribution

## Objectif
Personnaliser le branding de la distribution pour représenter la France et Emmanuel Macron.

## Étapes
1. **Changement du nom de la distribution**
   - Modifier le fichier `/etc/os-release` pour changer le nom en "MacronLinux".
   - Mettre à jour les fichiers de configuration pour refléter ce changement.

2. **Personnalisation de neofetch**
   - Modifier le fichier de configuration de neofetch pour afficher un logo personnalisé.
   - Utiliser un drapeau français comme logo.
   - Ajouter des informations sur Emmanuel Macron dans la sortie de neofetch.

3. **Ajout de références à la France**
   - Configurer le fond d'écran par défaut avec une image de Emmanuel Macron.
   - Ajouter des citations ou des messages inspirants dans les fichiers de configuration.
   - Personnaliser le terminal avec des couleurs bleu, blanc, rouge.

## Fichiers à Modifier
- `/etc/os-release`
- Configuration de neofetch.
- Fichiers de configuration du terminal.
