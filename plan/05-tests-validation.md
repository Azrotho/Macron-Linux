# Tests et Validation

## Objectif
Valider que la distribution fonctionne correctement et que toutes les personnalisations sont en place.

## Étapes
1. **Tests de l'image ISO**
   - Créer une machine virtuelle pour tester l'installation.
   - Vérifier que l'installation se déroule sans erreur.

2. **Validation des assets**
   - Vérifier que le fond d'écran est correctement appliqué.
   - Tester les sons (boot, erreur, etc.).
   - Valider que le thème France est appliqué.

3. **Validation du branding**
   - Vérifier que le nom de la distribution est "MacronLinux".
   - Tester neofetch pour voir le logo personnalisé.
   - Valider les couleurs et les références à la France.

4. **Tests de fonctionnalités**
   - Tester les applications KDE.
   - Vérifier la connexion et la déconnexion.
   - Tester les paramètres système.

## Critères de Validation
- L'image ISO s'installe sans erreur.
- Les assets (fond d'écran, sons) sont correctement intégrés.
- Le thème France est appliqué.
- Le branding (nom, logo, couleurs) est cohérent.
- Toutes les fonctionnalités de base fonctionnent.
